# Agent features:
- PC Model
- Hostname
- Local IP
- Monitor Model
- Service Tag (PC and Monitor)
- Windows version
- If windows is active
- CPU
- Drive Type and Size
- RAM
- Restart Computer if it's online for more three days
- Get all installed programs from computer
